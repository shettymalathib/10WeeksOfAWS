

## 1. What is a Region?

A Region is a geographic location where AWS has multiple Availability Zones and provides cloud services.

Example: Mumbai, Singapore, London.

## 2. What is an Availability Zone?

An Availability Zone (AZ) is one or more isolated data centers inside an AWS Region. Multiple AZs help keep applications available even if one AZ has a problem.

## 3. What is one thing AWS secures?

AWS secures the physical infrastructure, such as data centers, servers, networking, and hardware.

## 4. What is one thing you must secure?

You must secure your IAM users, passwords, applications, data, and permissions in your AWS account.


### Easy way to remember

| Question                      | Short Answer                                                   |
| ----------------------------- | -------------------------------------------------------------- |
| What is a Region?             | A geographic location containing multiple Availability Zones.  |
| What is an Availability Zone? | An isolated data center inside a Region.                       |
| What does AWS secure?         | The cloud infrastructure (data centers, hardware, networking). |
| What do you secure?           | Your data, applications, IAM users, and permissions.           |



> **AWS secures the cloud. You secure what you build in the cloud.**
